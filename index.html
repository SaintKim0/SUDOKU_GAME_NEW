        📋 기본 규칙

        스도쿠는 숫자 퍼즐 게임입니다. 다음 3가지 규칙을 모두 만족해야 합니다:

        • 행 규칙: 각 가로줄에 1부터 N까지의 숫자가 중복 없이 하나씩
        • 열 규칙: 각 세로줄에 1부터 N까지의 숫자가 중복 없이 하나씩
        • 박스 규칙: 각 박스에 1부터 N까지의 숫자가 중복 없이 하나씩

        🎯 게임 방법

        1. 빈 칸 선택: 채우고 싶은 빈 칸을 클릭합니다
        2. 숫자 입력: 하단의 숫자 버튼을 클릭하거나 키보드로 입력합니다
        3. 힌트 사용: 막힐 때는 힌트 버튼(?)을 사용하세요
        4. 정답 확인: 모든 칸을 채운 후 "정답 확인" 버튼을 누르세요
        5. 완료: 정답이면 축하 모달이 나타납니다!

        💡 게임 팁

        • 각 박스는 색상으로 구분되어 있어 규칙을 쉽게 확인할 수 있습니다
        • 빨간색 테두리는 잘못된 입력을 나타냅니다
        • 파란색 숫자는 미리 주어진 숫자입니다
        • 검은색 숫자는 사용자가 입력한 숫자입니다
        • 작은 보드(6×6)부터 시작해서 점진적으로 어려운 보드에 도전하세요
